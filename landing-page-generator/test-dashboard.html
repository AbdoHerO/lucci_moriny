Step-by-Step Testing Process

Step 1: Basic Setup
  1. Open the dashboard
  2. Enter project name: "اختبار النظام"
  3. Enter page title: "صفحة اختبار ديناميكية"
  4. Verify Arabic RTL display
Step 2: Content Configuration
  1. Navigate to "المحتوى والأسعار" tab
  2. Modify main title
  3. Change prices and currency
  4. Verify real-time updates
Step 3: Image Management
  1. Go to "الصور" tab
  2. Add multiple slider images
  3. Add body content images
  4. Test remove functionality
  5. Try both file upload and URL input
Step 4: Form Builder Testing
  1. Navigate to "النموذج" tab
  2. Change product type and observe form changes
  3. Add/remove color options
  4. Add/remove size options (for two variants)
  5. Toggle offer/gift option
Step 5: Reviews System
  1. Go to "التقييمات" tab
  2. Click "اختيار التقييمات"
  3. Select multiple reviews
  4. Confirm selection
  5. Test remove individual reviews
Step 6: Preview & Generation
  1. Click "معاينة" to test preview system
  2. Verify responsive preview
  3. Test device switching (desktop/tablet/mobile)
  4. Save configuration
  5. Generate and download ZIP file
